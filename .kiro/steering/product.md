# Product Overview

**Kiro Shmup** is an arcade-style side-scrolling game built as part of the AWS re:Invent workshop. The game serves as a collaborative learning experience where participants build games of their choice with AI assistance.

## Core Concept
A simple platformer/shmup hybrid featuring the Kiro logo as the player character. Players navigate a 2D environment, avoiding enemies while demonstrating basic game mechanics like movement, jumping, and collision detection.

## Target Audience
Workshop participants learning game development with AI assistance. Designed to be built incrementally over a 2-hour session, starting with minimal viable gameplay and adding features based on user preferences.

## Key Features
- Player movement and jumping mechanics
- Basic enemy collision
- Game state management (start, playing, game over)
- Kiro brand visual identity integration
- 60 FPS smooth gameplay
